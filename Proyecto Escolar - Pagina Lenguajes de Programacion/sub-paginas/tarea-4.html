Sintaxis

La sintaxis de Python tiene ciertas peculiaridades, la mas importante de estas es la sangria. En otros lenguajes de programación la sangría en el código es solo para facilitar la lectura, en Python la sangría es muy importante, se utiliza la sangría para indicar un bloque de código. Si llegaras a omitir una sangria, te saldria un error al ejecutar el archivo.

Otro de los aspectos importantes en la sintaxis de Python es que no utilizas un ";" para indicar el final de una instrucción. Python interpreta cada linea de codigo como una instrucción individual a menos que tenga sangria. 

if 5 > 2:
	print("Five is greater than two!")

A diferencia de otros lenguajes de programación, Python no tiene comandos para declarar una variable. Una variable se crea en el momento en que se le asigna un valor. 

x = 5
y = "John"

En Python las variables no necesitan ser declaradas con ningún tipo en particular e incluso pueden cambiar el tipo después de que se hayan establecido. 

x = 4 # x es de tipo entero
x = "Sally" # ahora x es de tipo string

Aplicación para Editar y Ejecutar

Para programar en Python se puede utilizar cualquier editor de texto plano, incluso el bloc de notas siempre y cuando el archivo se guarde con la extension ".py" Sin embargo se recomiendan editores mas avanzados que tengan remarcado de palabras clave a la hora de escribir codigo para tener una mejor experiencia al leer el codigo. Otro punto a tomar en cuenta antes de empezar a programar en Python es que se necesita descargar e instalar el lenguaje.

A que tipo de Paradigma Pertenece

Python es un lenguaje de programación multiparadigma. Esto significa que más que forzar a los programadores a adoptar un estilo particular de programación, permite varios estilos: programación orientada a objetos, programación imperativa y programación funcional, etc.

Ejemplo de programa

num = float(input("Ingresa un numero: "))
if num > 0:
	print("Numero Positivo")
elif num == 0:
	print("Cero")
else:
	print("Numero Negativo")
