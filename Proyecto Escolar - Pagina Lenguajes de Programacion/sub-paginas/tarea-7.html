Simbolos

Los operadores aritmeticos, comparativos y lógicos son muy importantes a la hora de programar funciones, ya que gracias a ellos se puede hacer distinciones de datos, agrupaciones, etc.. Se presenta a continuación una pequeña introducción de ellos:

Simbolo Significado

- Resta

/ División
^ Potencia

Sintaxis

En casi todos los demás lenguajes de programación, el signo igual asigna un cierto valor a una variable. Como ya sabemos, x = 3 significa que x ahora tiene el valor de 3. En R se hace de una manera diferente. El operador de asignación principal es <- como en: 

x <- 3

Una nota más sobre las variables: R es un lenguaje que distingue entre mayúsculas y minúsculas. Entonces, la variable x no es lo mismo que X. Eso se aplica a casi todo en R; por ejemplo, la función subconjunto() no es lo mismo que SubConjunto ().

Cuando creas un arreglo en R, tienes que agregar una pequeña parte mas al codigo. Para poner multiples valores a una sola variable se necesita la funcion c(). 

Mi-arrelgo <- c(1, 1, 2, 3, 5, 8)

Esqueleto de los Programas

R soporta como tipo de datos básicos valores numéricos, caracteres, cadenas de caracteres y valores booleanos. Los distintos tipos de datos pueden almacenarse en variables que se identifican con un nombre. Los nombres de variables deben comenzar por una letra seguida de más letras, dígitos o los caracteres punto (.) o subrayado (_). Se diferencia entre mayúsculas y minúsculas. Los nombres de las variables deben ser informativos utilizando por convenio él . Para separar distintas palabras. Para almacenar datos en una variable se utiliza el operador de asignación <- 

mi.variable.numerica <- 3.524
mi.variable.caracter <- "control"
mi.variable.booleana <- TRUE

Aplicación para Editar y Ejecutar

Junto con R se incluyen ocho bibliotecas o paquetes (llamadas bibliotecas estándar) pero otros muchos están disponibles a través de Internet. Actualmente se encuentran disponibles 9249 librerías (packages) desarrollados en R, que cubren multitud de aplicaciones en diversos campos. Una vez instalado el programa, la forma más fácil de usarlo es de forma interactiva mediante la línea de comandos. Cuando R se inicia, aparece la ventana del programa “Gui” (Graphical User Iinterface) con un mensaje de apertura.

A que tipo de Paradigma Pertenece

R es un lenguaje multiparadigma y cae en las categorias Funcional, Imperativo y Orientado a Objetos.

Ejemplo de programa

print(sample(1:3))

print(sample(c(2,5,3), size=4, replace=TRUE)
print(sample(1:2, size=10, prob=c(1,3), replace=TRUE))
