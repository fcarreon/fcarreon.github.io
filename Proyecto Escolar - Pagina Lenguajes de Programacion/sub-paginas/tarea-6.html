Simbolos

Haskell es un lenguaje funcional (donde todo se hace con llamadas a funciones), estático, implícitamente tipado (los tipos los revisa el compilador, pero no hace falta declararlos), perezoso (nada se hace hasta que es completamente necesario). Los lenguajes populares más parecidos son la familia ML (que no son, sin embargo, lenguajes perezosos).

Un programa de Haskell consiste en definiciones de funciones.

  • Declararla: indicar el tipo
  • Definirla: dar el metodo de computo

Haskell hace inferencia de tipos. Generalmente no es necesario declararlos a menos que se así se quiera. Si desea declarar tipos, se puede usar "::" para hacerlo. 

Prelude> :t True
True :: Bool
Prelude> :t 'X'
'X' :: Char
Prelude> :t "Hello, Haskell"
"Hello, Haskell" :: [Char]

En Haskell algunas cosas se tornan más interesantes con los números. 

Prelude> :t 42
42 :: (Num t) => t
Prelude> :t 42.0
42.0 :: (Fractional t) => t
Prelude> :t gcd 15 20
gcd 15 20 :: (Integral t) => t

Estos tipos usan "Clases de tipo". Significan:

  • 42 se puede usar como cualquier tipo numérico. (Esta es la razón por la que podemos declarar 5 como Int o como Double).
  • 42.0 puede ser cualquier tipo fraccionario, pero no un tipo integral.
  • gcd 15 20 (que es una llamada de función) puede ser cualquier tipo integral, pero no un tipo fraccionario.

El simbolo final es un (), que se pronuncia "unidad". Solo tiene un valor, que se escribe también como () y se pronuncia "unidad" de 

Prelude> ()
()
Prelude> :t ()
() :: ()

Se puede pensar en este tipo como en void de la familia de lenguajes de C. Se puede retornar () desde una acción de I/O cuando no se quiere retornar nada en particular.

Sintaxis

Expresiones Simples

Es posible escribir varias expresiones matemáticas directamente en ghci y obtener un resultado. Prelude es el prompt por defecto de GHCi. 

Prelude> 3 * 5
15
Prelude> 4 ^ 2 - 1
15
Prelude> (1 - 5)^(3 * 2 - 4)
16

Las cadenas van entre "comillas dobles." Puede concatenarlas con "++" 

Prelude> "Hello"
"Hello"
Prelude> "Hello" ++ ", Haskell"
"Hello, Haskell"

Las funciones se llaman colocando los argumentos directamente después del nombre de la función. No hay paréntesis en la llamada a la función: 

Prelude> succ 5
6
Prelude> truncate 6.59
6
Prelude> round 6.59
7
Prelude> sqrt 2
1.4142135623730951
Prelude> not (5 < 3)
True
Prelude> gcd 21 14
7

Esqueleto de los Programas

En el lenguaje Haskell el esqueleto de un programa utiliza la definición de acciones I/O, llamada main: 

main = do putStrLn "What is 2 + 2?"
	x <- readLn
	if x == 4
		then putStrLn "You're right!"
		else putStrLn "You're wrong!"

Aplicación para Editar y Ejecutar

El compilador de Haskell más común es GHC. Los ejecutables de GHC están disponibles para GNU/Linux, FreeBSD, MacOS, Windows, y Solaris. Una vez instalado GHC, se tienen dos programas que son de interés ahora: ghc, y ghci. El primero compila librerías y aplicaciones escritas en Haskell a código binario. El segundo es un interprete que permite escribir código Haskell y obtener un resultado inmediato.

A que tipo de Paradigma Pertenece

Haskell pertenece al paradigma funcional de programación.

Ejemplo de programa

module Main where
 
factorial n = if n == 0 then 1 else n * factorial (n - 1)
 
main = do putStrLn "What is 5! ?"
	x <- readLn
	if x == factorial 5
		then putStrLn "You're right!"
		else putStrLn "You're wrong!"
